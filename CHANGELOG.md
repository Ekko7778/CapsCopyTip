# Changelog

All notable changes to CapsCopyTip will be documented in this file.

## [1.4.0] - 2026-03-04

### Added
- ✨ 引入 DefaultConfig 统一管理默认配置

### Changed
- 🖼️ 设置窗口 UI 重构：粗体 Text + 分割线替代 GroupBox
- 🖼️ 优化设置窗口布局和功能开关排列
- 🖼️ 添加 GitHub 图标和版权信息
- ♻️ 优化代码注释

### Fixed
- 🐛 修复 IME 中/英文检测逻辑

## [1.3.1] - 2026-03-03

### Fixed
- 🐛 彻底修复设置窗口内存泄漏问题
- ⚡ 优化 IME 检测和剪贴板处理性能

## [1.3.0] - 2026-03-03

### Added
- ✨ 内置 CaretIndicator 光标指示器（替换外部 exe）
- ✨ 添加"显示中/英状态"开关
- 🖼️ 添加应用图标

### Changed
- ⚡ 优化 Shift 键触发逻辑，排除组合键干扰
- 🖼️ 优化设置窗口布局
- ♻️ 整理项目目录结构

### Fixed
- 🐛 使用正确的 Stop() 方法停止光标指示器

## [1.2.1] - 2026-03-02

### Fixed
- 🐛 添加剪贴板防抖机制，修复重复显示
- 🐛 修正输入法中英文显示状态反转

## [1.2.0] - 2026-03-02

### Added
- ✨ 添加浅色模式支持
- ✨ 添加顶部和底部提示偏移设置
- ✨ 添加"反转输入法检测"选项
- ✨ 添加文本光标语言标记功能
- ✨ 设置界面添加 GitHub 链接
- 🔄 添加恢复默认设置按钮
- ⚙️ 添加鼠标偏移距离设置

### Changed
- ⚡ 优化输入法检测逻辑，适配 Windows 11 微软拼音
- ⚡ 优化 Shift 切换响应速度
- 🖼️ 优化大小写提示显示格式

## [1.1.0] - 2026-03-02

### Added
- ✨ 添加托盘菜单（设置、重启、退出）
- ✨ 添加设置窗口（功能开关、显示时长、提示位置、字体样式）
- ✨ 添加配置文件持久化（config.ini）
- 🖥️ 添加开机启动功能

### Changed
- 🏷️ 项目重命名为 CapsCopyTip（原 AllInOneNotification）

## [1.0.1] - 2026-03-01

### Fixed
- 🐛 修复提示窗口闪烁问题
- 🐛 修复跟随鼠标时位置偏移问题
- 🐛 修复 IME 状态查询权限错误

## [1.0.0] - 2026-03-01

### Added
- ✨ 大小写提示功能（CapsLock 状态检测）
- ✨ 输入法中/英状态显示
- ✨ 复制提示功能（字符数/图片/文件数）
